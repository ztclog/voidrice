# 安装
- 复制喜欢的主题目录到 ~/.local/share/fcitx5/themes/ 目录下。
- 打开 fcitx5配置，选择 配置附加组件，经典用户界面栏点击配置，选择对应主题即可。

# 遇到的问题
- [ssfconv](https://github.com/fkxxyz/ssfconv)工具转换成fcitx5主题有时候会有一点点样式问题，自己稍加调整即可。
